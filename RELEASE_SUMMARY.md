## 🚀 快速开始

```bash
git clone https://github.com/kangkona/py-mono.git
cd py-mono
pip install -e ".[dev]"
./scripts/install-dev.sh

# Web UI
export OPENAI_API_KEY=your-key
py-webui

# Coding Agent
py-code
```
